# Aerotoolbox

# ABOUT

**English**

Aerotoolbox is a comprehensive web application designed for aerospace engineering students. It serves as a digital toolbox for performing essential aerodynamic and atmospheric calculations.

Key features include:
- **ISA Calculator**: Calculates atmospheric properties (Temperature, Pressure, Density, Speed of Sound) based on the International Standard Atmosphere (ISA) model validation up to the tropopause.
- **Aerodynamics**: Tools to calculate Lift and Drag coefficients (CL, CD, Cdi, Cdp) and other parameters like Wing Area, Aspect Ratio, and more.
- **Unit Conversion**: Built-in support for converting between metric and imperial units for all calculations.

**Español**

Aerotoolbox es una aplicación web integral diseñada para estudiantes de la carrera Ingeniería Aeroespacial. Sirve como una caja de herramientas digital para realizar cálculos aerodinámicos y atmosféricos esenciales.

Las características principales incluyen:
- **Calculadora ISA**: Calcula propiedades atmosféricas (Temperatura, Presión, Densidad, Velocidad del Sonido) basadas en el modelo de la Atmósfera Estándar Internacional (ISA) hasta la tropopausa.
- **Aerodinámica**: Herramientas para calcular coeficientes de Sustentación y Resistencia (CL, CD, Cdi, Cdp) y otros parámetros como Superficie Alar, Alargamiento, etc.
- **Conversión de Unidades**: Soporte integrado para convertir entre unidades métricas e imperiales para todos los cálculos.
